# FonTakip API (Core Banking Simulation)

Bu proje, kurumsal bir bankanın "Hazine Uygulamaları" departmanındaki süreçlerden ilham alınarak geliştirilmiş bir yatırım fonu takip ve yönetim sisteminin arka plan (Backend) mimarisidir.

## 🚀 Proje Hakkında
Mikroservis mimarisine hazırlık ve modern yazılım geliştirme standartlarını kavramak amacıyla başlatılan bu proje; yatırım fonlarının fiyat değişimlerinin takibi ve kullanıcı portföy yönetimini simüle etmektedir. 

**Not:** Bu proje, modern yazılım geliştirme pratiklerini yakalamak ve öğrenme eğrisini hızlandırmak amacıyla yapay zeka (AI) destekli bir çalışma metodolojisi ile çift taraflı (Pair-Programming) geliştirilmektedir. Mimari kararlar ve kodlama süreçleri AI mentörlüğü ile desteklenmektedir.

## 🛠 Kullanılan Teknolojiler
* **Framework:** .NET 8.0 / ASP.NET Core Web API
* **Veritabanı:** Microsoft SQL Server (Docker Container)
* **ORM:** Entity Framework Core
* **Mimari Yaklaşım:** OOP, Dependency Injection, RESTful API
